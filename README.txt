Run "python -m venv .venv" tp initialize the virtual environment.
Attached is the requirements.txt file which contains the libraries needed for the project.
Run "pip install -r requirements.txt" to install them.
To run web application, run 'source .venv/bin/activate' to run the virtual environment
Go to /doctor_list and run 'python manage runserver'
Go to /doctor_list/frontend and install all the dependencies in the package.json file.
Run 'npm run serve'
   - This should bring up a server running localhost:8080
